## Problem

[올바른 괄호](https://programmers.co.kr/learn/courses/30/lessons/12909)

---

## Idea

(를 0, )를 1이라고 생각해보면  
`())(()` = `011001`  
위와 같이 표현할 수 있다.

0과 다음 0사이에 1이 0의 갯수보다 크다면 올바른 괄호가 아니다.  
이 점을 이용해서 문제를 풀었다.

---

## Code

```cpp
#include<string>

using namespace std;

bool solution(string s)
{
    int a = 0;
    int b = 0;
    bool answer = true;

    for(int i=0; i<s.length(); i++){
        if( a < b ){
            answer = false;
            break;
        }

        if( s[i] == '(')    a++;
        else                b++;
    }

    if( a != b){
        answer = false;
    }

    return answer;
}
```
