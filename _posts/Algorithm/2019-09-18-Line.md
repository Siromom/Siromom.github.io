## Idea

**순열 문제**

1. 1~n까지 vector에 넣는다.
2. next_permutation을 k번 반복

👉 k는 n! 이하의 자연수이기 때문에 시간복잡도가 O(n!)이 된다.

## Code

1. 첫번째 시도: 시간복잡도 O(n!)

```cpp
#include <vector>
#include <algorithm>

using namespace std;

vector<int> solution(int n, long long k) {
    vector<int> num;
    for(int i=1; i<=n; i++){
        num.push_back(i);
    }

    int count = 1;
    while(count < k){
        count++;
        next_permutation(num.begin(), num.end());

    }

    return num;
}
```
