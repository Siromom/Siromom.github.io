## Problem

[Baekjoon 11866](https://www.acmicpc.net/problem/11866)

## Idea

1. 1부터 n까지 수를 큐에 집어넣는다.
2. 큐가 empty가 될 때까지 반복문 실행
3. 1~k까지 카운트 해서 카운트 값이 k가 아니라면 pop했던 값을 다시 push 해주고, k라면 그냥 pop 한다.

---

## Code

```cpp
#include <iostream>
#include <queue>

using namespace std;

int main() {
	int n, k;
	cin >> n >> k;

	queue<int> q;

	for (int i = 1; i <= n; i++) {
		q.push(i);
	}

	int count = 1;
	cout << "<";
	while (!q.empty()) {
		int first = q.front();
		q.pop();

		if (count == k) {
			cout << first;
			count = 1;

			if (!q.empty()) {
				cout << ", ";
			}
		}
		else {
			q.push(first);
			count++;
		}
	}
	cout << ">\n";

	return 0;
}

```
