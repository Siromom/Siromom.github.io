## Problem

[Baekjoon 2675](https://www.acmicpc.net/problem/2675)

## Idea

1. 문자열, 반복 횟수 입력
2. 문자열의 첫번째 문자부터 마지막 문자까지 돌면서 반복횟수만큼 출력하는 2중 for문 작성

---

## Code

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {

	int t;
	cin >> t;

	while (t--) {
		// 반복 횟수 입력
		int r;
		cin >> r;

		// 문자열 입력
		string text;
		cin >> text;

		for (int i = 0; i < text.size(); i++) {
			for (int j = 0; j < r; j++) {
				cout << text[i];
			}
		}
		cout << endl;
	}

	return 0;
}
```

## Review
