## Problem

[Baekjoon 2941](https://www.acmicpc.net/problem/2941)


## Idea

단순 무식하게 각 문자마다 크로아티아 알파벳 8개 케이스인지 아닌지 따지는 방식으로 구했다.


---

## Code

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void)
{

    string in;
    cin >> in;

    int sum = 0;
    for (int i = 0; i < in.length(); i++)
    {

        switch (in[i])
        {
        case 'c':
            if (in[i + 1] == '=' || in[i + 1] == '-')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        case 'd':
            if (in[i + 1] == 'z' && in[i + 2] == '=')
            {
                sum += 1;
                i += 2;
                break;
            }
            else if (in[i + 1] == '-')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        case 'l':
            if (in[i + 1] == 'j')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        case 'n':
            if (in[i + 1] == 'j')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        case 's':
            if (in[i + 1] == '=')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        case 'z':
            if (in[i + 1] == '=')
            {
                sum += 1;
                i++;
                break;
            }

            sum += 1;
            break;

        default:
            sum += 1;
            break;
        }
    }

    cout << sum << endl;

    return 0;
}
```

## Review
이런 경우에 switch case로 작성해주는 것보다 case들을 배열에 집어넣고 for문을 돌리는게 코드가 더 깔끔하게 나올 수 있을 것 같다.
