## Problem

[Baekjoon 9461](https://www.acmicpc.net/problem/9461)

## Idea

수열을 나열하다 보면 규칙성을 발견할 수 있다.

P(1) = 1    
P(2) = 1    
P(3) = 1      
P(4) = 2     
P(5) = 2     
P(6) = 3     
P(7) = 4    
P(8) = 5      
P(9) = 7    
P(10) = 9     
   .    
   .    
   .    

`P(N) = P(N-1) + P(N-5)` (N > 5)      
이 규칙성을 이용해 P(N)을 구한다.

---

## Code

```cpp
#include <iostream>

using namespace std;

long long int p[101];

int main(void)
{

    int t;
    cin >> t;

    p[1] = p[2] = p[3] = 1;
    p[4] = 2;

    while (t--)
    {
        int n;
        cin >> n;

        if (p[n] != 0)
        {
            cout << p[n] << endl;
            continue;
        }

        for (int i = 5; i <= n; i++)
        {
            p[i] = p[i - 1] + p[i - 5];
        }
        cout << p[n] << endl;
    }

    return 0;
}
```
