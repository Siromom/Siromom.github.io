## Idea

재귀 대표문제        
- 피보나치
- 하노이의 탑


---

## Code
```cpp
#include <iostream>

using namespace std;

int fibo(int n)
{
    if (n <= 1)
    {
        return n;
    }

    return fibo(n - 1) + fibo(n - 2);
}

int main()
{
    int n;
    cin >> n;
    cout << fibo(n) << endl;
    return 0;
}
```
