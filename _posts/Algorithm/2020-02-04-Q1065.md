## Problem

[Baekjoon 1065](https://www.acmicpc.net/problem/1065)


## Idea

`한수 = 각 자리수들이 등차수열을 이루는 수`

1자리수, 2자리수는 비교대상이 없거나, 하나밖에 없으므로 모두 한수가 된다.
3자리수부터 한수인지 아닌지 판단한다.

1. 10으로 나눈 나머지를 배열에 넣고, 10으로 나누는 과정을 반복해서 각 자리수를 배열에 집어 넣는다.
2. `첫째자리수 - 둘째자리수`와 `둘째자리수 - 셋째자리수` 계산 값이 같으면 한수, 다르면 한수가 아니다.
3. N의 최댓값이 1000이고, 1000은 한수가 아니므로 999까지의 경우의 수만 고려하면 된다.

## Code
```cpp
#include <iostream>
#include <vector>

using namespace std;

int getNum(int n){
	vector<int> nums;
	while(n > 0){
		nums.push_back(n%10);
		n /= 10;
	}
	
	if(nums[0]-nums[1] == nums[1]-nums[2]){
		return 1;
	}
	else {
		return 0;
	}
}

int main() {
	
	int n;
	scanf("%d", &n);
	
	int ans = 0;
	if(n <= 99){
		ans = n;
	}
	else if(n == 1000){
		ans = 99;
		for(int i=100; i<1000; i++){
			if(getNum(i) != 0)	ans++;
		}
	}
	else {
		ans = 99;
		for(int i=100; i<=n; i++){
			if(getNum(i) != 0)	ans++;
		}	
	}
	
	printf("%d\n", ans);
	return 0;
}
```
