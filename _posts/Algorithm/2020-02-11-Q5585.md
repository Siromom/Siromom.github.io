## Problem

[Baekjoon 5585](https://www.acmicpc.net/problem/5585)

## Idea

계산해야 할 돈이 500원 > 100원 이런식으로 불규칙적이기 때문에 배열에 따로 담아놓았다.        
그리고 chk 배열에는 해당 지폐를 몇 장 썼는지 (나누기 값) 저장한다.

---

## Code
```cpp
#include <iostream>

using namespace std;

int money[6] = { 500, 100, 50, 10, 5, 1 };
int chk[6];

int main(int argc, char** argv) {
	
	int n;
	scanf("%d", &n);
	
	n = 1000 - n;
	
	int i = 0;
	while(n>0){
		if(n / money[i] > 0){
			chk[i] = n / money[i];
		}
		n %= money[i]; 
		i++;
	}
	
	int ans = 0;
	for(int i=0; i<6; i++){
		if(chk[i])	ans+=chk[i];
	}
	printf("%d\n", ans);
	return 0;
}
```
