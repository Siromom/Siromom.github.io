## 순열 - 재귀적 방법으로 구현

```java
import java.util.Scanner;

public class Basic_permu {

	static int[] input = { 1, 2, 3, 4, 5 };
	static int[] numbers;
	static boolean[] isUsed;
	static int r;

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		r = sc.nextInt();

		numbers = new int[r]; 				// 순열을 저장하는 배열
		isUsed = new boolean[input.length];	// 사용 체크하는 배열

		permu(0);
		sc.close();
	}

	public static void permu(int cnt) {

		// 순열 완성
		if (cnt == r) {
			printArray(numbers);
			return;
		}

		for(int i=0; i<input.length; i++) {
			if(!isUsed[i]) {
				isUsed[i] = true;
				numbers[cnt] = input[i];
				permu(cnt + 1);
				isUsed[i] = false;
			}
		}
	}

	public static void printArray(int[] arr) {
		for (int i = 0; i < arr.length; i++) {
			System.out.print(arr[i] + " ");
		}
		System.out.println();
	}
}
```

## 조합 - 재귀적 방법으로 구현

```java
import java.util.Scanner;

/**
 * n개 중 r개 뽑기
 * */

public class Basic_comb {

	static int[] input = {1, 2, 3, 4, 5};
	static int[] numbers;
	static int r;
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		r = sc.nextInt();

		numbers = new int[r];	// 조합을 저장하는 배열

		comb(0, 0);
        sc.close();
	}

	public static void comb(int cnt, int cur) {

		// 조합 완성
		if(cnt == r) {
			printArray(numbers);
			return;
		}

		if(cur == input.length)	return;


		numbers[cnt] = input[cur];
		comb(cnt+1, cur+1);
		comb(cnt, cur+1);
	}

	public static void printArray(int[] arr) {
		for(int i=0; i<arr.length; i++) {
			System.out.print(arr[i] + " ");
		}
		System.out.println();
	}
}
```

## 순열 - NextPermutation

```java
import java.util.Arrays;

public class NextPermutation {


	public static void main(String[] args) {

		int[] input = { 1, 2, 3, 4, 5 };

		// 1. 정렬
		Arrays.sort(input);

		do {
			printArray(input);
		}while(np(input));

	}

	public static boolean np(int[] input) {

		// 1. 꼭대기 찾기
		int i= input.length-1;
		while(i > 0 && input[i-1] > input[i]) {
			--i;
		}

		if(i == 0)	return false;

		// 2. i-1위치와 교환할 j 찾기
		int j = input.length-1;
		while(input[i-1] > input[j]){
			--j;
		}

		// 3. 바꾸기
		input = swap(input, i-1, j);

		// 4. i부터 오름차순 정렬
		int k = input.length-1;
		while(i<k) {
			input = swap(input, i++, k--);
		}

		return true;
	}

	public static int[] swap(int[] input, int i, int j) {
		int tmp = input[i];
		input[i] = input[j];
		input[j] = tmp;
		return input;
	}

	public static void printArray(int[] arr) {
		for (int i = 0; i < arr.length; i++) {
			System.out.print(arr[i] + " ");
		}
		System.out.println();
	}
}

```

1. 맨뒷자리부터 꼭대기 지점 i 찾고 (i가 0이 되면 마지막 순열)
2. 꼭대기 바로 앞 부분 i-1과 바꿀 위치를 맨 뒷자리에서 찾는다. (input[i-1]보다 큰 가장 작은 수)
3. i-1 값과 j 값 교환
4. i 위치부터 맨 뒤까지 오름차순 정렬

## 조합 - NextPermutation

```java
import java.util.Arrays;

public class Comb_NP {

	public static void main(String[] args) {
		int[] input = { 1, 2, 3, 4, 5 };

		int r = 3;
		int[] comb = new int[input.length];
		for(int i=0; i<r; i++) {
			comb[i] = 1;
		}

		// 정렬
		Arrays.sort(comb);

		do {
			int index = 0;
			int[] combArr = new int[r];

			for(int i=0; i<comb.length; i++) {
				if(comb[i] == 1) {
					combArr[index++] = input[i];
				}
			}

			printArray(combArr);
		} while (np(comb));
	}

	public static boolean np(int[] input) {
		int i = input.length-1;
		while(i > 0 && input[i-1] >= input[i]) {
			--i;
		}
		if(i == 0)	return false;

		int j = input.length-1;
		while(input[i-1] >= input[j]) {
			--j;
		}

		input = swap(input, i-1, j);

		int k = input.length-1;
		while(i<k) {
			input = swap(input, i++, k--);
		}
		return true;
	}

	public static int[] swap(int[] input, int i, int j) {
		int tmp = input[i];
		input[i] = input[j];
		input[j] = tmp;
		return input;
	}

	public static void printArray(int[] arr) {
		for (int i = 0; i < arr.length; i++) {
			System.out.print(arr[i] + " ");
		}
		System.out.println();
	}
}

```

아무래도 재귀적 방법이 더 편하다.  
특히나 조합을 Next Permutation으로 구현하는 건 더 어렵다 ㅠ  
아직 익숙하지 않아서 그런걸까...  
속도면에서는 재귀를 사용하지 않는 next permutation이 빠르기 때문에 그 구현방법도 꼭 익혀야하는데 더 자주 복습해야겠다.

## 부분집합 - 재귀적 방법으로 구현

```java
public class Basic_subset {

	static int[] input = { 1, 2, 3 };
	static boolean[] isSelected;
	static int N = 3;
	public static void main(String[] args) {

		isSelected = new boolean[N];

		subSet(0);
	}

	public static void subSet(int cnt) {
		if(cnt == N) {
			System.out.print("부분집합: ");
			for(int i=0; i<N; i++) {
				if(isSelected[i]) {
					System.out.print(input[i]);
				}
			}
			System.out.println();
		}

		if(cnt == N) return;

		isSelected[cnt] = true;
		subSet(cnt+1);

		isSelected[cnt] = false;
		subSet(cnt+1);
	}

}

```

## 부분집합 - 비트마스크로 구현

```java
public class Subset_bit {

	public static void main(String[] args) {

		int[] input = { 1, 2, 3 };
		int N = input.length;

		// n=3 일경우 경우의 수 2^3=8
		// 0~7 값을 비트마스킹 활용
		for(int flag=0; flag<(1<<N); flag++) {
			System.out.print("부분집합: ");

			// 원소 수만큼 비트 비교
			for(int j=0; j<N; j++) {
				// flag의 j번째 비트가 1이면 원소 출력
				System.out.print((flag & (1<<j)) != 0 ? input[j] : "");
			}
			System.out.println();
		}

	}

}
```
