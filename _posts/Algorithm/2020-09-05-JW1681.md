tsp(외판원 순회) 문제

## Idea

1. `순열`로 모든 경우의 수를 구한다.
2. 출발지는 항상 1번이기 때문에 1로 시작하는 순열이 아니면 가지치기

---

## Code

### 재귀적 방법으로 순열 구현 - 💣시간초과

```java
import java.util.Scanner;

public class algo {

	static int N, min;
	static int[][] arr;
	static int[] input;
	static boolean[] visited;
	public static void main(String[] args) {
		// Input
		Scanner sc = new Scanner(System.in);
		N = sc.nextInt();				// 배달해야하는 장소의 수
		arr = new int[N][N];	// 인접행렬
		for(int i=0; i<N; i++) {
			for(int j=0; j<N; j++) {
				arr[i][j] = sc.nextInt();
			}
		}
		min = Integer.MAX_VALUE;

		// 순열을 만들기 위한 배열
		input = new int[N];
		visited = new boolean[N];
		for(int i=0; i<N; i++) {
			input[i] = i;
		}

		perm(0);

		// Output
		System.out.println(min);
		sc.close();
	}

	private static void perm(int cnt) {

		// 기저조건
		if(cnt == N) {

			if(input[0] == 0) {		// 출발지점이 0일 때만 비용 계산
				int sum = 0;
				int last = 0;	// 마지막 방문한 도시 (0부터 방문시작하므로 초기값 0)
				for(int i=1; i<N; i++) {
					sum += arr[last][input[i]];
					last = input[i];
				}
				sum += arr[last][0];		// 출발점으로 돌아오는 비용
				min = Math.min(min, sum);	// 최솟값 갱신
			}
			return;
		}

		// 유도 조건
		for(int i=0; i<N; i++) {
			if(!visited[i]) {
				visited[i] = true;
				input[cnt] = i;
				perm(cnt+1);
				visited[i] = false;
			}
		}

	}

}

```

### next permutation으로 순열 구현 - 💣시간초과

```java

import java.util.Scanner;

public class algo_np {

	static int N, min;
	static int[] input;
	public static void main(String[] args) {
		// Input
		Scanner sc = new Scanner(System.in);
		N = sc.nextInt();				// 배달해야하는 장소의 수
		int[][] arr = new int[N][N];	// 인접행렬
		for(int i=0; i<N; i++) {
			for(int j=0; j<N; j++) {
				arr[i][j] = sc.nextInt();
			}
		}
		min = Integer.MAX_VALUE;

		// 순열을 만들기 위한 배열
		input = new int[N];
		for(int i=0; i<N; i++) {
			input[i] = i;
		}

		do {
			if(input[0] == 0) {
				// 출발지점이 0이면 비용 계산
				int sum = 0;
				int last = 0;	// 마지막 방문한 도시 (0부터 방문시작하므로 초기값 0)
				for(int i=1; i<N; i++) {
					sum += arr[last][input[i]];
					last = input[i];
				}
				sum += arr[last][0];		// 출발점으로 돌아오는 비용
				min = Math.min(min, sum);	// 최솟값 갱신
			}
		}while(nextPermutation());


		// Output
		System.out.println(min);
		sc.close();
	}

	private static boolean nextPermutation() {
		int i = N-1;
		while( i>0 && input[i-1] > input[i] ) {
			--i;
		}
		if(i==0)	return false;

		int j = N-1;
		while(input[i-1] > input[j]) {
			--j;
		}

		swap(i-1, j);

		int k = N-1;
		while(i<k) {
			swap(i++, k--);
		}

		return true;
	}

	private static void swap(int i, int j) {
		int tmp = input[i];
		input[i] = input[j];
		input[j] = tmp;
	}
}

```

## 순열로 풀면 시간초과가 난다?!

재귀적 방법으로 푸는것, next permutation 모두 💣시간초과...

시간복잡도가 O(N!)이 나오기 때문에 순열로는 해결할 수 없다.

## 백트래킹 ✅PASS✅

모든 경우의 수를 계산하지 말고, 백트래킹을 사용해서 start 지점을 0으로 정해놓은 경우만 살펴본다.

```java

import java.util.Scanner;

public class algo {

	static int N, min;
	static int[][] arr;
	static int[] visited;
	public static void main(String[] args) {
		// Input
		Scanner sc = new Scanner(System.in);
		N = sc.nextInt();		// 배달해야하는 장소의 수
		arr = new int[N][N];	// 인접행렬
		for(int i=0; i<N; i++) {
			for(int j=0; j<N; j++) {
				arr[i][j] = sc.nextInt();
			}
		}
		min = Integer.MAX_VALUE;
		visited = new int[N];
		visited[0] = 1;

		tsp(0, 0, 0, 1);

		// Output
		System.out.println(min);
		sc.close();
	}

	private static void tsp(int start, int location, int sum, int cnt) {
		if(cnt == N) {

			if(arr[location][start] == 0)	return;	// 다시 시작한 곳으로 돌아갈 수 없는 경우

			sum += arr[location][start];
			min = Math.min(min, sum);
			return;
		}

		for(int i=0; i<N; i++) {
			if(visited[i] == 0 && arr[location][i] != 0) {
				visited[i] = 1;
				tsp(start, i, sum+arr[location][i], cnt+1);
				visited[i] = 0;
			}
		}
	}
}

```
