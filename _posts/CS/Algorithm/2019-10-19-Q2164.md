## Problem

[Baekjoon 2164](https://www.acmicpc.net/problem/2164)

## Idea

게임 진행조건
1. 맨 위 카드 버리기
2. 그 다음 카드 맨 아래로 옮기기

맨 앞에서 pop하고, 맨 뒤에 push하는 방식이므로 deque를 이용하면 쉽게 구현할 수 있다.

---

## Code

```cpp
#include <iostream>
#include <queue>

using namespace std;

int main(void)
{
    int n;
    cin >> n;

    deque<int> dq;
    for (int i = 1; i <= n; i++)
    {
        dq.push_back(i);
    }

    while (dq.size() != 1)
    {
        dq.pop_front();
        int temp = dq.front();
        dq.pop_front();
        dq.push_back(temp);
    }

    cout << dq.front();

    return 0;
}
```

## Review

그냥 deque이 생각나서 이걸로 푸니까 쉽고 간단하게 풀렸다. 문제 난이도 하! 자료구조만 알고 있으면 되는 문제!
