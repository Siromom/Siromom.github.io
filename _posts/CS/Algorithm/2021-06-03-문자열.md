# 테크닉

### String to Int

```java
String str = "123";
int ret = Integer.parseInt(str);
```

### Char to Int

```java
char ch = "1";
int ret = ch - '0';
```

### Int to String

```java
int num = 123;
Strin ret = Integer.toString(num);
```

### Int to Char

```java
int num = 13;
Strin ret = (char)(num + 'A');
```
