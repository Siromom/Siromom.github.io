# 연산자 ==와 ===의 비교

== : Equal Operator     
=== : Strict Equal Operator

== 는 자료형과 상관없이 내용만 비교하는 것이고, === 는 자료형까지 비교하는 것이다.

```javascript
1 == "1"     --> true
1 === "1"    --> false
```
