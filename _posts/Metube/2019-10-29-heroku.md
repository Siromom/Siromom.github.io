# deploying

1. Build server for production
    - src 폴더안의 모든 파일을 build 폴더에 build
    - webpack static 파일이 build 폴더에 생기도록 설정 변경
2. Upload to Heroku
    1. Install Heroku    
    2. Heroku login
    3. `heroku create` 실행
    4. 실행 후 나온 git remote 주소를 복사해서 그 주소로 push
    5. Fix errors
